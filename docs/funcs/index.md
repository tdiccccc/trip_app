# 機能仕様一覧

## 概要

伊勢旅行メモリーアプリの全機能一覧。
各機能の実装状況と対応するページ・API を記載する（最終更新: 2026-03-11）。

---

## 機能詳細

### 1. ログイン

- Sanctum SPA 認証（Cookie ベース）
- CSRF トークン取得 → メール + パスワードでログイン
- 認証済みユーザーは `/` へ自動リダイレクト
- ログアウト機能あり（`useAuth` composable 経由）

### 2. カウントダウン

- 旅行日（2026-03-28）までの日・時・分・秒をリアルタイム表示
- 1秒間隔で更新
- 旅行日を過ぎた場合は「Have a great trip!」メッセージを表示
- しおり・アルバムへのクイックリンク

### 3. しおり（タイムライン）

- 日付タブで Day 1 / Day 2 を切り替え
- 行程を時系列で一覧表示（sort_order 順）
- モーダルフォームから行程の追加・編集
- 各行程の削除（確認ダイアログ付き）
- 当日は現在時刻に該当する行程を自動ハイライト（1分間隔更新）

### 4. スポット詳細

- ヒーロー画像表示（画像がない場合はプレースホルダー）
- カテゴリ表示（観光・グルメ・宿泊・その他）
- 住所・営業時間・料金情報のカード表示
- Google Maps への外部リンク
- スポットに紐づく写真のグリッド表示
- メモの一覧表示・新規追加

### 5. 写真アルバム

- 写真のグリッド表示（PhotoGrid コンポーネント）
- 写真アップロード（PhotoUploader コンポーネント、spot_id / caption / taken_at 指定可能）
- スポット別フィルタリング（スポット一覧を API から取得）
- 写真タップで全画面 Lightbox 表示（インライン実装）
- Lightbox 内から写真削除

### 6. スライドショー

- 全画面レイアウトで写真を自動再生
- 撮影日時順（asc）で写真を取得
- トランジション切り替え（フェード / スライド）
- 再生速度の調整
- 再生・一時停止・前へ・次へのコントロール
- コントロール UI は3秒後に自動非表示、画面タップで再表示
- 写真がない場合のエンプティステート表示

### 7. ふたりの掲示板

- テキストメッセージの投稿
- 投稿一覧表示（新しい順、自分の投稿は右寄せ）
- 絵文字リアクション機能（6種類のプリセット）
- リアクション数の集計表示（自分が付けたリアクションはハイライト）
- 投稿フォームは画面下部に固定配置

### 8. パッキングリスト

- チェックボックス形式の持ち物リスト
- 担当者フィルタリング（全て / たろう / はなこ）
- 準備状況のプログレスバー表示
- アイテムの新規追加（名前 + 担当者選択）
- チェック状態のトグル（チェック済みは半透明 + 取り消し線）
- アイテムの削除（確認ダイアログ付き）

### 9. 費用メモ

- 支出の新規追加（項目名・金額・カテゴリ・支払者・メモ）
- カテゴリ: 食事・交通・チケット・お土産・宿泊・その他
- 支払者: たろう / はなこ
- サマリーカード（合計金額・一人あたり・カテゴリ別内訳）
- カテゴリ別フィルタリング
- 支出の削除（確認ダイアログ付き）

### 10. エクスポート

- 現状はスタブ（タイトルと説明文のみ）
- バックエンド API は PDF しおり・フォトブック PDF・ZIP ダウンロードに対応済み
- フロントエンドの UI 実装が今後必要
